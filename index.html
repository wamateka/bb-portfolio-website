Internship

Last summer, of 2023, I Interned at Media box, one of the best tech companies in Burundi, as part of a Work experience Program at school. For 3 months, not only did I learn, but I was also able to apply the different knowledge I had in different technologies. Starting the first month by producing functional specification for different projects at Media box, in French, a language I did not master, allowed me to understand how projects are designed in big Enterprises. In the second Month, I delved into Iot. Finally a more technical job for me. I was not really familiarized with electronics, but playing around with the different IoT devices broadened my knowledge on how this technologies can be used to make life better and more efficient. I conceived a small security system using ultrasonic sensor and Arduino, which were programed using c++. Just to say a few. I ended the Internship with A hands on experience on a green-house automation project where I contributed in programming stepper motors to lift the plastic sheeting used to cover the green House to allow more air to get in during hot weather
